# README

SONAERUは自然災害や感染症などのリスクが高まる状況の中、これらのリスクに対する情報共有の場として活用することを目的としております。 

![トップページ](images/Top.png "top")
![グラフ機能](images/colona.png "colona")


# 開発環境

* Ruby 2.7.0

* Ruby on Rails 5.2.4.3

* MySQL 8.0.19

* JavaScript

* BootStrap

* Github

* Rspec

* Docker

* Nginx

* Unicorn

* AWS(EC2,RDS)


# 実装した機能一覧及
* ログイン機能(簡単ログイン機能)

* フォロー機能

* コロナの最新感染情報をグラフで表示する機能(chart.js)

* 星レビュー機能

* 楽天API機能

* 記事投稿機能

* ページネーション機能

* タグ付機能

* 画像プレビュー機能
