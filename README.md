# README

SONAERUは自然災害や感染症などのリスクが高まる状況の中、これらのリスクに対する情報共有の場として活用することを目的としております。 
それに伴い、防災意識の向上が見込まれ、自然災害や感染症のリスクへの逓減にも大きな役割を果たせます。

**実装した機能一覧**
* ログイン機能

* フォロー機能

* コロナの最新感染情報をグラフで表示する機能

* 星レビュー機能

* 楽天API機能

* 記事投稿機能

* ページネーション機能

* タグ付機能

* 画像プレビュー機能
